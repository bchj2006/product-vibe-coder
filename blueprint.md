# 음식 메뉴 랜덤 추천 웹 앱 (Food Vibe Recommender)

현대적이고 감각적인 UI를 가진 음식 메뉴 랜덤 추천 웹 애플리케이션입니다.

## 프로젝트 개요
- **목적:** 결정 장애를 겪는 사용자들에게 시각적으로 즐거운 경험과 함께 메뉴를 추천합니다.
- **주요 기능:**
    - "메뉴 추천받기" 버튼을 통한 랜덤 메뉴 생성.
    - 현대적인 CSS 애니메이션과 효과.
    - 반응형 레이아웃 (Mobile & Desktop).

## 기술 스택 및 디자인 가이드
- **HTML:** Semantic markup, Web Components 사용 고려.
- **CSS:** CSS Variables, Container Queries, `:has()` selector, OKLCH 컬러 시스템, 네이티브 유리 모피즘(Glassmorphism) 효과.
- **JavaScript:** ES Modules, DOM Manipulation, CSS Web Animations API.
